# quickviews
Add-on para Blender 2.79

### Cómo instalar el add-on quickviews en Blender 2.79

1. Abre Blender 2.79.

2. Ve al menú superior izquierdo y haz clic en File (Archivo) > User Preferences (Preferencias de usuario).

3. En la ventana que se abre, haz clic en la pestaña Add-ons (ubicada en la barra superior).

4. En la parte inferior de esa ventana, haz clic en el botón Install App from File... (Instalar aplicación desde archivo...).

5. Busca en tu computadora el archivo .py de tu add-on (o el archivo .zip si lo comprimiste) y selecciónalo.

6. Haz clic en el botón superior derecho Install App from File... para confirmar.

7. En el buscador de la izquierda, escribe quickviews para encontrarlo.

8. Activa el add-on marcando la casilla de verificación (el cuadrado del lado izquierdo de su nombre).

9. **Muy importante:** Haz clic en el botón Save User Settings (Guardar configuraciones de usuario) en la esquina inferior izquierda. Si no lo haces, el add-on se desactivará la próxima vez que abras Blender.
